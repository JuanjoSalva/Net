A. if (!(counter % 25)) B. if (counter == 25) C. if (counter >> 25 == 0) D. if (counter << 25 == 0) Respuesta: A - % (Módulo) Calcula el resto entero de dividir 2 números. Cada 25, (25, 50, 75, 100...), el resto de divir entre 25 es 0. Incorrecto: counter == 25. Funciona la primera vez porque el numero de registros procesados es 25 y se muestra.
