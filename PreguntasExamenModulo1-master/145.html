Pregunta 145. objetivo de la pregunta: Está desarrollando una página web que incluye el siguiente HTML. <span id="myTextSpan" class = "redText">Hello There!</span> El color de fuente del texto dentro del intervalo debe ser rojo. Necesita desarrollar la página web para cumplir con el requisito. ¿Qué dos segmentos de código logran el objetivo? Cada respuesta correcta presenta una solución completa. NOTA: Cada selección correcta vale un punto. A. span#myTextSpan{ color:red; } B. .redText{ color:red; } C. #myTextSpan1{ color:red; } D. #redText{ color:red } Respuesta: ABC El primero sí vale porque coge elementos span con id = "myTextSpan" El segundo sí vale porque coge elemento con clase "redText" como es el nuestro. El tercero sí vale porque coge elemento con id "myTextSpan" como es el nuestro. El cuarto no vale porque coge elemento con id "redText" y no hay ninguno.
